🏗️ Module 2 — Architecture Google Cloud

Architecture du projet

Notre pipeline de données utilisera plusieurs services Google Cloud. 

CSV Files
    ↓
Cloud Storage
    ↓
BigQuery (RAW)
    ↓
DBT Transformations
    ↓
Analytics Tables

Services utilisés

  • Cloud Storage
    Stockage des fichiers de données (Data Lake)
  • BigQuery
    Data Warehouse serverless pour l’analyse des données
  • DBT
    Outil de transformation des données dans BigQuery
  • Terraform
    Gestion de l’infrastructure cloud avec Infrastructure as Code
