☁️ Module 1 — Fondamentaux Cloud & Serverless

1️⃣ C’est quoi le Cloud Computing ?

Le Cloud Computing consiste à utiliser des ressources informatiques via Internet au lieu d’installer et gérer ses propres serveurs.

Exemples de ressources cloud :

  • Serveurs
  • Stockage
  • Bases de données
  • Services de calcul

Les principaux fournisseurs de cloud sont :

  • Google Cloud Platform (GCP)
  • AWS
  • Microsoft Azure

2️⃣ C’est quoi le Serverless ?

Le serverless signifie que les développeurs ne gèrent pas les serveurs. Le fournisseur cloud s’occupe de l’infrastructure.

Avantages :

  • Pas de gestion des serveurs
  • Scalabilité automatique
  • Paiement uniquement à l’utilisation

Exemples de services serverless :

  • Google BigQuery
  • Cloud Functions
  • Cloud Run

3️⃣ ETL vs ELT

Deux méthodes pour traiter les données :

ETL

Extract → Transform → Load

Les données sont transformées avant d’être chargées dans le Data Warehouse.

ELT

Extract → Load → Transform

Les données sont d’abord chargées dans le Data Warehouse (ex: BigQuery) puis transformées avec des outils comme DBT.
