Module 1 – Fondamentaux Cloud & Serverless

1️⃣ C’est quoi le Cloud Computing ?

Le Cloud Computing consiste à utiliser des ressources informatiques (serveurs, stockage, bases de données) via Internet au lieu de les installer physiquement chez soi.

2️⃣ C’est quoi le Serverless ?

Serverless signifie que tu ne gères pas les serveurs. Le cloud provider s’en occupe. Tu payes uniquement à l’usage.

3️⃣ ETL vs ELT

  • ETL : Transform avant chargement
  • ELT : Charger puis transformer (BigQuery + DBT)
