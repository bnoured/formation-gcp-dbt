🚀 Formation GCP & DBT

📘 Projet : End-to-End Serverless ELT Pipeline

Architecture du Pipeline ELT

Le pipeline de données suit cette architecture :

Retail CSV Files
      ↓
Cloud Storage (Data Lake)
      ↓
BigQuery RAW Dataset
      ↓
DBT Transformations
      ↓
Analytics Tables

Technologies utilisées :

  • Google Cloud Storage — stockage des fichiers
  • BigQuery — Data Warehouse
  • DBT — transformation des données
  • Terraform — gestion de l’infrastructure

Objectif : Construire un pipeline ELT serverless sur Google Cloud Platform avec DBT.
