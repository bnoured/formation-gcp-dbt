⚙️ Module 4 — ELT Pipeline (Ingestion)

Objectif

Construire un pipeline d’ingestion pour charger les données retail dans BigQuery.

Architecture du pipeline

Retail CSV Files
        ↓
Cloud Storage (Data Lake)
        ↓
BigQuery RAW Dataset

Étapes du pipeline

  • Uploader les fichiers CSV dans un bucket Google Cloud Storage
  • Créer un dataset BigQuery RAW
  • Charger les données dans BigQuery
