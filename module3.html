📦 Module 3 — Dataset Retail

Objectif

Créer un dataset retail qui sera utilisé pour construire notre pipeline ELT.

Structure du dataset retail

Le projet utilise plusieurs fichiers CSV représentant un système de vente :

  • customers.csv — informations clients
  • products.csv — catalogue produits
  • orders.csv — commandes
  • retail_sales.csv — lignes de vente

Source des données

Les données seront stockées sous forme de fichiers CSV dans Google Cloud Storage.
